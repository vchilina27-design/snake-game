# Snake Game

Простая игра "Змейка", написанная на Python с использованием библиотеки Pygame.

## Требования
- Python 3.x
- Pygame

## Установка
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите игру:
   ```bash
   python snake.py
   ```

## Управление
- Стрелки на клавиатуре для движения.
- **Q** — выйти из игры.
- **C** — играть снова после проигрыша.
