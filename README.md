# Snake Game

Простая игра "Змейка", написанная на Python с использованием библиотеки Pygame.

## Требования
- Python 3.x
- Pygame

## Установка
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите игру:
   ```bash
   python snake.py
   ```

## Управление
- Стрелки на клавиатуре для движения.
- **Q** — выйти из игры.
- **C** — играть снова после проигрыша.

## Бонусы

### Остановка Времени (Time Stop)
- Фиолетовый ромб появляется на карте каждые **10 секунд**.
- Когда змейка съедает его, игра замедляется на **5 секунд**.
- Во время действия эффекта змейка становится голубой, а на экране отображается индикатор оставшегося времени.
- После окончания эффекта скорость возвращается к нормальной, и через 10 секунд появляется новый бонус.
